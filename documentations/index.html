Firebase Setup #back to top

STEP 1: Create New Firebase Database

Go to following links https://console.firebase.google.com then create new database.

STEP 2: Press Continue Button

Google Analytics is a free and unlimited analytics solution that enables targeting, reporting, and more in Firebase Crashlytics, Cloud Messaging, In-App Messaging, Remote Config, A/B Testing, Predictions, and Cloud Functions.

STEP 3: Check all of requirements , then click on Create Project

Upon project creation, a new Google Analytics property will be created and linked to your Firebase project. This link will enable data flow between the products. Data exported from your Google Analytics property into Firebase is subject to the Firebase terms of service, while Firebase data imported into Google Analytics is subject to the Google Analytics terms of service. Learn more.

STEP 4: Enable Email/Password Sign in Method , Then press Save Button

Allow users to sign up using their email address and password. Our SDKs also provide email address verification, password recovery, and email address change primitives

STEP 6: Enable G-Mail Sign in Method

Google sign-in is automatically configured on your connected iOS and web apps. To set up Google sign-in for your Android apps, you need to add the SHA1 fingerprint for each app on your Project Settings.

STEP 7: CREATE FIRESTORE DATABASE

Choose Firebase Realtime Database and Create new Database, Then choose test mode.

STEP 8: CHOOSE START IN TEST MODE and Press Enable Button

Once you have defined your data structure you will have to write rules to secure your data.

STEP 9: CHOOSE FIRESTORE LOCATION

ENTER TEST INFO into FIREBASE REALTIME DATABASE , Then Press on Add button

STEP 10: CHANGE FIRESTORE RULES

Go to Firebase Storage Section and click on Next button

STEP 11: ENTER TEST INFO

ENTER TEST INFO into FIRESTORE DATABASE , Then Press on Next button

STEP 12: Enable Firebase Storage

Go to Firebase Storage Section and click on Next button

STEP 13: Set Cloud Storage Location

Select Nearest location from you and click on Done button

																service firebase.storage {
																	   match /b/{bucket}/o {
